[**View Live Demo**](https://forever-ever-brand-guideline.vercel.app)

---

## ✨ Features

- **Interactive Design Specs:** Fluid, slider-based interactivity for visualizing spacing and border radius metrics.
- **Dynamic Grid Anatomy:** Visualizations of the `Router = Rinner + P` geometry.
- **Responsive Layouts:** Carefully crafted for both desktop and mobile experiences.
- **Premium Aesthetics:** High-quality visual shadows, depth effects, and haptic-like responsive elements.
- **Vite Powered:** Lightning-fast HMR and optimized production builds.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/nestorwwh-hue/forever-ever-brand-guideline.git
   ```

2. Navigate to the project directory:
   ```bash
   cd forever-ever-brand-guideline
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

## 🛠️ Tech Stack

- **Framework:** Vanilla JavaScript + HTML/CSS
- **Bundler:** [Vite](https://vitejs.dev/)
- **Deployment:** [Vercel](https://vercel.com/)
